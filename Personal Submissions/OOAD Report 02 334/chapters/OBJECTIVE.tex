\chapter{OBJECTIVE}\label{obj}


The primary objective of this lab is to understand and model the dynamic behavior of software systems by translating textual requirements into UML Sequence Diagrams. Specifically, this lab aims to:
\begin{itemize}
    \item Identify key actors, system participants, and objects from detailed scenario descriptions.
    \item Map out the chronological flow of interactions and messages between these entities.
    \item Represent object lifelines and execution occurrences (activations) accurately.
    \item Gain practical experience using PlantUML syntax to generate professional software design documentation for various real-world systems.
\end{itemize}
