\chapter{EQUIPMENT USED}
\label{ch:eqp_used}
This chapter describes the equipment and resources used to conduct the laboratory activity on exploring UML drawing tools and preparing a comparative team report. The listed equipment supported research, diagram creation, collaboration, communication, and documentation throughout the experiment.\\

The following equipment and resources were used to complete the activity:

\begin{itemize}
  \item Computer or laptop with internet access for researching UML tools and creating diagrams.
  \item Web-based UML drawing tools such as Draw.io, Lucidchart, PlantUML, Diagrams.net, and SmartDraw.
  \item Diagram creation software (Draw.io desktop or browser version) for preparing sample UML diagrams.
  \item Microsoft Word or Google Docs for writing and formatting the laboratory report.
  \item Presentation and collaboration platforms such as Google Drive for file sharing and storage.
  \item Trello board for task management and tracking team progress.
  \item Discord (or similar communication platform) for team discussion and coordination.
  \item Smartphone or screenshot tools for capturing evidence of team collaboration and contributions.
  \item Cloud storage services (Google Drive/Dropbox) for backup and safe storage of documents and diagrams.
\end{itemize}
