UML Tool Review: Draw.io
-------------------------
-->Introduction: UML (Unified Modeling Language) is a visual language used in software engineering to design and understand systems before development. Common UML diagrams include class diagrams, use case diagrams, sequence diagrams, and activity diagrams. To create these diagrams, tools like Draw.io (diagrams.net) are widely used.

-->Overview: Draw.io is a free online and offline diagram tool that helps users create UML diagrams, flowcharts, ER diagrams, and network diagrams. It runs in a web browser and is popular among students and professionals because it is easy to use and accessible.

-->Key Features: 1. Simple drag-and-drop interface
                 2. Built-in UML templates (class, use case, sequence, activity diagrams)
                 3. Free and works on all operating systems
                 4. Supports cloud storage (Google Drive, OneDrive, Dropbox)
                 5. Allows collaboration with others
                 6.Export diagrams as PDF, PNG, JPG, SVG, or XML

-->Advantages: 1. Completely free
               2. Beginner-friendly
               3. No account required
               4. Works online and offline
               5. Many UML shapes available

-->Limitation: 1. Needs internet for online use
                  2. Fewer advanced features than professional tools
                  3. Large diagrams may slow down

-->Conclusion: Draw.io is a simple, free, and effective UML tool for students and beginners. It provides useful templates, easy editing, and multiple export options, making it a great choice for creating UML diagrams for academic work.
