\subsection*{TANVIR ALAM SHIHAB}

My primary contribution to this project was the design and specification of the system's core security and architectural frameworks. I focused on the following key areas:

\begin{itemize}
    \item \textbf{Functional Requirement 01 (Vendor Onboarding \& Security):} 
    I authored the requirements for \textit{Vendor Registration, Subscription, and Secure Authentication}. This involved defining the workflow for verifying legal business documents (E-TIN, Trade License), integrating mandatory subscription payments during sign-up, and specifying the logic for Two-Factor Authentication (2FA) to secure vendor accounts.

    \item \textbf{Functional Requirement 02 (Multi-Tenant Architecture):} 
    I developed the specifications for \textit{Multi-Tenant Data Isolation and Security}. I defined the "Database-per-Tenant" architectural pattern to ensure strict data segregation between competing vendors. I also wrote the user stories for automated database provisioning and the centralized management of tenant data to prevent cross-tenant data leakage.
\end{itemize}
