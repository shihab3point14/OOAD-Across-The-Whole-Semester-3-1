\chapter{OBJECTIVE}\label{obj}


The main objectives of this lab are:
\begin{enumerate}
    \item To understand the concept and importance of user stories in software development.
    \item To learn how to write user stories using a standard format.
    \item To identify different system users and their requirements.
    \item To map user stories to the functional requirements of an e-commerce system.
    \item To improve clarity and simplicity in requirement documentation.
\end{enumerate}
