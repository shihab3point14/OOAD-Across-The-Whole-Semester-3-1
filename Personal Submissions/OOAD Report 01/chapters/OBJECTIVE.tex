\chapter{OBJECTIVE}\label{obj}


\section{Objectives}
This lab aimed to:

\begin{itemize}
    \item See how work moves step-by-step using columns on a board.
    \item Practice working together as a team and handing out jobs.
    \item Get better at doing more than one task at a time when possible.
    \item Check each other's work, especially the calculations.
    \item Keep a full record using pictures and written notes.
    \item Look at how the team worked and figure out what could be done better next time.
\end{itemize}


\section{Expected Outcomes}

\begin{itemize}
    \item Successful completion of all nine assigned tasks with documented evidence
    \item Identification of tasks suitable for parallel execution
    \item Implementation of a peer testing mechanism for quality assurance
    \item Understanding of team member roles and responsibilities
    \item Recognition of challenges in team communication and task clarity
    \item Development of insights for improving collaborative workflow management
\end{itemize}
