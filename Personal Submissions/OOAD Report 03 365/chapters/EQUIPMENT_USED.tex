\chapter{EQUIPMENT USED}
\label{ch:eqp_used}

The following tools and resources were utilized to prepare and complete this Software Requirements Specification (SRS) document:

\begin{enumerate}
    \item A personal computer or laptop with internet access.
    \item Web browsers (e.g., Google Chrome, Mozilla Firefox, Brave) for research and collaboration.
    \item Word processing and text editing software (e.g., Notepad, text editors, MS Word, or \LaTeX) for drafting and formatting the SRS.
    \item Software Requirements Specification (SRS) guidelines and templates provided through Google Classroom.
    \item Discord server for team communication, discussion, and coordination.
    \item Trello for task management, progress tracking, and project organization.
    \item Course materials related to Object-Oriented Analysis and Design (OOAD) for reference and requirement analysis.
\end{enumerate}
