\chapter{EQUIPMENT USED}
\label{ch:eqp_used}

The following tools and resources were used to complete this lab:

\begin{itemize}
    \item \textbf{Modeling Tool:} sequenceDiagram.org (Web-based UML sequence diagram modeling tool)
    
    \item \textbf{Methodology:} Unified Modeling Language (UML) 2.x standards for Sequence Diagrams to represent system interactions and message flow.
    
    \item \textbf{Documentation Tools:} LaTeX / Markdown for preparing and formatting the lab report.
    
    \item \textbf{Hardware:} Personal Computer with internet browser access.
    
    \item \textbf{Reference Materials:} System scenarios and requirements describing the School Attendance System, Springfield University Enrollment System, Online Shopping System, and University Exam Process.
\end{itemize}
