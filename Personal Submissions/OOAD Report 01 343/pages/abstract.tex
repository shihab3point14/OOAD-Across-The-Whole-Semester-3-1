This laboratory exercise simulated agile project management using a paper-based Kanban system to coordinate team tasks. Through a series of nine predetermined activities—ranging from data collection to arithmetic calculations—team members practiced workflow visualization, parallel task execution, role assignment, and peer verification processes. Using sticky notes organized into four columns (\textit{To Do}, \textit{Doing}, \textit{Test}, \textit{Done}), the team completed tasks while documenting each state transition. The exercise highlighted the importance of clear communication, task delegation, and process transparency in collaborative environments. Key outcomes included successful completion of all tasks with peer-verified calculations, identification of workflow efficiencies through parallel execution, and recognition of challenges related to data consistency and task ambiguity. This report documents the methodology, results, and insights gained from implementing a simple yet effective visual management system.
